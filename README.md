# 📄 Resume Screening System 

## 🎯 Project Overview

This project is an AI-powered Resume Screening System that helps recruiters automatically:
- Analyze resumes
- Match resumes with job descriptions
- Rank candidates based on relevance
- Extract skills from resumes
- Identify missing skills
- Provide analytics on resume dataset

It reduces manual effort in hiring and improves the speed of candidate shortlisting.

---

## 🚀 Features

- 📄 Resume text preprocessing
- 🔍 Job description matching using NLP
- 📊 TF-IDF + Cosine Similarity scoring
- 🏆 Candidate ranking system
- 🧠 Skill extraction from resumes
- ❌ Missing skill identification
- 📊 Interactive analytics dashboard (Streamlit)
- 👤 Candidate details viewer
- 📁 Multi-page Streamlit application

---

## 🛠️ Tech Stack

- Python 🐍
- Streamlit
- Pandas
- Scikit-learn
- Matplotlib
- Regex (re)
